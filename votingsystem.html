ONLINE VOTING SYSTEM

Brief Description:
The Ondo State Youth Network Online Voting System is a secure platform designed to facilitate electronic voting for youth elections. The system includes an admin portal for managing user registrations, monitoring real-time voting results, and setting automated start and end times for voting. Registered voters can participate securely, ensuring that each voter can only cast a single vote during the election period.

Your Role:
I independently developed the entire application, managing both front-end and back-end development. This involved creating the user interface for voters and administrators, as well as implementing the logic for secure voting, user authentication, and real-time vote counting.

Challenges & Solutions:

  • Challenge 1:
    • Challenge:

      • Ensuring that voting could only occur within the predefined date and time was crucial for maintaining the integrity of the election.

    • Solution:

      • I implemented a JavaScript timer that controlled the opening and closing of the voting session. This timer automatically activated and deactivated the voting functionality, providing a seamless and automated voting experience for users.

Outcome/Impact:
The voting system was successfully deployed for the Ondo State Youth Network, allowing a large number of users to securely cast their votes online. The system effectively prevented multiple voting attempts and provided real-time results to the admin, contributing to a transparent and efficient election process.
